Кодирование:
В ячейку "Ввод сообщения" нужно ввести трехзначное число в двоичной системы счисления.
В ячейки для порождающей матрицы нужно вводить по одному символу в ячейку.
После ввода, выбрать функцию "Кодировать сообщение" и нажать на кнопку "Операция".
В поле "Ответ" будет выведено кодовое слово.

Декодирование:
В ячейку "Ввод сообщения" нужно ввести шестизначное число в двоичной системы счисления.
В ячейки для порождающей матрицы нужно вводить по одному символу в ячейку.
После ввода, выбрать функцию "Декодировать сообщение" и нажать на кнопку "Операция".
В поле "Ответ" будет показано раскодированное слово.
В нижнем поле будет показана таблица смежных классов.
Для демонстрации процесса декодирования нужно нажать кнопку "Показать декодирование".
